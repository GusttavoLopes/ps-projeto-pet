#### Run

#expo-cli global instalado - $ npm i -g expo-cli

$ git clone https://github.com/GusttavoLopes/ps-projeto-pet.git

$ cd pet

$ yarn install

$ expo install ou npm install

$ npm start (yarn web ou npm run web)

porta:19006 - http://localhost:19006/
